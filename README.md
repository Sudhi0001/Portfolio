# ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Sudhi0001/Portfolio.git
```

Navigate to project folder:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
